## 📋 Resumen Ejecutivo

El proyecto **APP (Arquitectura de Procesos y Plataforma) Chía-Girardot** estructura e implementa los **Sistemas Inteligentes de Transporte (ITS)** en el corredor vial Chía-Mosquera-Tocaima-Girardot (306 km), bajo la modalidad **Brownfield con enfoque de quinta generación**.

| Parámetro | Valor |
|-----------|-------|
| **Longitud corredor** | 306 km (158 km arteriales + 148 km colectores) |
| **Unidades Funcionales (UF)** | 6 UF + CCO + Telecomunicaciones |
| **Tramos** | 15 tramos principales |
| **Subtramos** | 93 subtramos |
| **Sectores** | 6 |
| **EOV** | 15 escenarios operativos |
| **CC (Componentes de Campo)** | 42 componentes (CC-01 a CC-42) |
| **Dispositivos totales** | ~634 |
| **Inversión estimada** | $11.5M — $19.3M USD |
| **ROI** | 3-5 años |
| **Modelo Financiero** | V6 — 20 hojas, 15 tramos, 93 subtramos, 373.7 km |
