## 📋 Resumen Ejecutivo

El proyecto **APP (Arquitectura de Procesos y Plataforma) Chía-Girardot** tiene como objetivo estructurar e implementar los Sistemas Inteligentes de Transporte (ITS) en el corredor vial Chía-Mosquera-Tocaima-Girardot (306 km), bajo la modalidad **Brownfield con enfoque de quinta generación**.

| Parámetro | Valor |
|-----------|-------|
| **Longitud corredor** | 306 km |
| **Unidades funcionales** | ~85 (incluye CCO, patios, emergencias) |
| **Sectores** | 6 |
| **EOV** | 15 escenarios operativos |
| **CC** | 42 componentes de campo |
| **Dispositivos totales** | ~634 |
| **Inversión estimada** | $11.5M — $19.3M USD |
| **ROI** | 3-5 años |
