# PLAN DE GESTIÓN DEL PROYECTO — APP CHÍA-GIRARDOT

## 1. ENFOQUE DE GESTIÓN

Metodología: PM2 (Project Management Methodology) — Comisión Europea  
Sistema de Soporte: ZEUS EOS  
Ciclo de Vida: 4 fases (Inicio, Planificación, Ejecución, Cierre)

## 2. PROCESOS PM2 APLICADOS

| Proceso | Descripción | Aplicación al Proyecto |
|---------|-------------|------------------------|
| Gestión de Alcance | Definir y controlar el trabajo | 5 entregables ITS, 42 componentes |
| Gestión de Calendario | Planificar y controlar tiempos | Cronograma de 5 entregables |
| Gestión de Costos | Estimar y controlar presupuesto | Oferta firmada, control de horas |
| Gestión de Calidad | Asegurar cumplimiento de estándares | Normativa ITS, Res. 28675/2022 |
| Gestión de Stakeholders | Comunicación y expectativas | TransConsult, ICCU, interventoría |
| Gestión de Riesgos | Identificar y mitigar amenazas | Técnicos, contractuales, legales |
| Gestión de Cambios | Controlar modificaciones | Versión 1.0, cambios con aprobación DP |
| Gestión de Adquisiciones | Contratar recursos externos | TransConsult como cliente |
