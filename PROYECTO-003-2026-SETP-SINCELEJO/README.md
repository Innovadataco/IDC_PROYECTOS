## 📋 Resumen Ejecutivo

El **Sistema Estratégico de Transporte Público (SETP) de Sincelejo** es un proyecto de modernización del transporte público en la ciudad de Sincelejo, Sucre, Colombia. El objetivo es implementar un sistema integral de gestión del transporte público que incluya planificación, operación, control y tarificación.
