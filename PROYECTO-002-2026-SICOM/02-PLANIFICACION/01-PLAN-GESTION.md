# PLAN DE GESTIÓN DEL PROYECTO — EMPALME SICOM

## 1. ENFOQUE DE GESTIÓN

Metodología: PM2 Innovadataco v1.0  
Sistema de Soporte: ZEUS EOS  
Ciclo de Vida: 8 fases (Preparación, Inventario, Acceso, Transferencia, Replicación, Corte, Operación, Transferencia Final)

## 2. PROCESOS PM2 APLICADOS

| Proceso | Descripción | Aplicación al Empalme |
|---------|-------------|------------------------|
| Gestión de Alcance | Controlar el trabajo del empalme | Inventario, accesos, pruebas, corte |
| Gestión de Calendario | Planificar y controlar tiempos | 6 semanas fijas |
| Gestión de Costos | Estimar y controlar presupuesto | Empalme $713M, Operación $2.728M |
| Gestión de Calidad | Asegurar cumplimiento ANS | 99.90%, RPO/RTO, MTTD/MTTR |
| Gestión de Stakeholders | Comunicación y expectativas | MME, INTERNEXA, CI2 |
| Gestión de Riesgos | Identificar y mitigar amenazas | 12 riesgos identificados |
| Gestión de Cambios | Controlar modificaciones | Gestión formal con aprobación MME |
| Gestión de Adquisiciones | Contratar recursos | IND TECH ↔ Innovadataco |
