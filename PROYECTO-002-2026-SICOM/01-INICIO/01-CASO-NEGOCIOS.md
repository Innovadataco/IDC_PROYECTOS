# CASO DE NEGOCIO — EMPALME SICOM

## 1. JUSTIFICACIÓN ESTRATÉGICA

El SICOM es un sistema crítico para Colombia. Garantizar su continuidad operativa durante la transición de operador es una obligación del Estado y una oportunidad de negocio para CI2.

## 2. ANÁLISIS DE COSTO-BENEFICIO

| Concepto | Valor |
|----------|-------|
| **Inversión empalme (6 semanas)** | ~$713M COP |
| **Inversión operación (5 meses)** | ~$2.728M COP |
| **Disponibilidad requerida** | 99.90% |
| **Usuarios iniciales** | 25,000 |
| **Usuarios proyectados** | 45,000 |
| **Transacciones** | 600/min web services, 300/min aplicación |
| **Multa por incumplimiento ANS** | 0.5% facturación por cada 0.1% debajo de 99.90% |

## 3. ALINEACIÓN CON OBJETIVOS DEL CLIENTE (MME)

- Continuidad operativa del SICOM sin interrupciones
- Cumplimiento de ANS estrictos
- Transferencia ordenada de operador
- Soberanía tecnológica y de datos

## 4. RIESGOS DEL NO-PROYECTO

- Interrupción del SICOM = crisis energética nacional
- Pérdida de datos de combustibles
- Sanciones por incumplimiento contractual
- Daño reputacional para el MME
