## 2. SISTEMAS PRINCIPALES

### 2.1 Sistema de Recaudo Centralizado (SRC)

| Componente | DescripciГіn | EspecificaciГіn |
|:-----------|:------------|:---------------|
| **Validadores** | Lectura medios pago | Mifare Desfire, QR, EMV sin contacto Nivel 1-2, GPS, 4G, pantalla 7", IP54 |
| **Motor tarifas** | CГЎlculo tarifas | Zonal, temporal, fare capping, transferencias |
| **Clearing** | DistribuciГіn ingresos | AutomГЎtico entre operador, ente gestor, recaudo |
| **CompensaciГіn** | Reglas negocio | DefiniciГіn actores, porcentajes, retenciones |
| **Medios pago** | Formas de pago | Tarjeta fГӯsica, app QR, NFC, efectivo (registro electrГіnico) |
| **Listas blancas/negras** | Control acceso | En tiempo real, centralizado |
| **OperaciГіn offline** | Contingencia | ValidaciГіn local, sincronizaciГіn posterior |

### 2.2 Sistema de GestiГіn y Control de Flota (SGCF)

| Componente | DescripciГіn | EspecificaciГіn |
|:-----------|:------------|:---------------|
| **GPS/GNSS** | Posicionamiento | Вұ10m, tiempo real, historial 12 meses |
| **Control itinerarios** | Rutas, frecuencias | AsignaciГіn, despacho, cumplimiento |
| **Alertas operativas** | Eventos crГӯticos | Exceso velocidad, fuera de ruta, botГіn pГЎnico, detenciГіn prolongada |
| **Videovigilancia** | Seguridad | 2 cГЎmaras 1080p por bus, audio, grabaciГіn 30 dГӯas |
| **Conteo pasajeros** | Demanda | CГЎmaras por puerta, fiabilidad вүҘ 97%, bidireccional |
| **Consola conductor** | InformaciГіn operativa | Pantalla 7" tГЎctil, rugerizada, IP54, 50В°C, entradas, ocupaciГіn, recorrido, tiempos |
| **BotГіn pГЎnico** | Emergencia | Conductor + pasajeros, alerta CCO, policГӯa, audio/video |
| **Reportes** | AnГЎlisis | PDF/Excel exportable, pasajeros por ruta, ingresos, km recorridos |

### 2.3 Sistema de InformaciГіn al Usuario (SIU)

| Componente | DescripciГіn | EspecificaciГіn |
|:-----------|:------------|:---------------|
| **App mГіvil** | iOS/Android | Planificador viajes, tiempos reales, recarga PSE/bancos, QR, NFC |
| **Micrositio web** | PГЎgina web | Recarga, consulta saldo, planeaciГіn, PQRS |
| **Pantallas paraderos** | Tiempo real | Tiempos llegada, ocupaciГіn, incidencias |
| **Predictor** | EstimaciГіn | ML/AI para tiempos llegada basado en trГЎfico |
| **WiFi gratuito** | Conectividad | Buses (opcional, bajo acuerdo comercial) |
| **Call center** | AtenciГіn | LГӯnea directa, WhatsApp, PQRS |

### 2.4 Sistema de Seguridad de Transacciones (SST)

| Componente | DescripciГіn | EspecificaciГіn |
|:-----------|:------------|:---------------|
| **SAM/HSM** | MГіdulos seguridad | SAM AV2 (70 unidades) o HSM servidor |
| **Cifrado** | ProtecciГіn datos | End-to-end, TLS/SSL, AES-256 |
| **AutenticaciГіn** | Identidad | OAuth2, tokens, certificados digitales |
| **GestiГіn claves** | Cryptography | GeneraciГіn, distribuciГіn, rotaciГіn, revocaciГіn |
| **Certificaciones EMV** | Cumplimiento | Nivel 1 y 2 en validadores |
| **Transferencia llaves** | Propiedad | Mapping completo a Metro Sabanas |
| **AuditorГӯa** | Seguridad | Logs inmutables, pentesting anual, cumplimiento Ley 1581/2012 |
